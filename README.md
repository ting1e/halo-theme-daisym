## 配置



### 社交资料

填写 github/wechat 等社交链接，正确填写会在首页有对应图标显示

### 链接

基于 https://github.com/halo-sigs/plugin-links

### 主题设置

* 默认开启局部刷新

## 页面

主题配置中，侧边栏选项可以选择是否开启各个子块
